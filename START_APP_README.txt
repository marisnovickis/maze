On Windows
1) Open CMD window and go to the place where mazeSolver.jar is located
for example, 
	cd "C:\Root\maze\"
2) Test that java is installed. Run 
	java –version
You should get something like this
	java version "1.8.0_77"
	Java(TM) SE Runtime Environment (build 1.8.0_77-b03)
	Java HotSpot(TM) 64-Bit Server VM (build 25.77-b03, mixed mode)
If Java runtime environment is not installed go here and install it 
	http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html
3) In command line type the following
	java -jar mazeSolver.jar "sparse_medium.txt
